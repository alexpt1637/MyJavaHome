Элементы для ввода даты и времени

        Для работы с датами и временем в HTML5 предназначено несколько типов элементов input:

        datetime-local: устанавливает дату и время
        date: устанавливает дату
        month: устанавливает текущий месяц и год
        time: устанавливает время
        week: устанавливает текущую неделю

        Например, используем поле для установки даты:

        label for="firstname">Имя: <
        input type="text" id="firstname" name="firstname"

        label for="date">Дата рождения: <
        input type="date" id="date" name="date"

        button type="submit">Отправить<

        И при вводе в поле для даты будет открываться календарик:
    

        Действие этого элемента зависит от браузера. В данном случае используется Google Chrome.
        В последних версиях Opera элемент не будет сильно отличаться. А вот в Microsoft Edge элемент будет выглядеть иначе.
        А браузеры Firefox и IE11 вообще не поддерживают данный элемент и отображают его как обычное текстовое поле.

        Применение остальных элементов:

        label for="week">Неделя: <
        input type="week" name="week" id="week"

        label for="localdate">Дата и время: <
        input type="datetime-local" id="localdate" name="date"

        label for="month">Месяц: <
        input type="month" id="month" name="month"

        label for="time">Время: <
        input type="time" id="time" name="time"

        button type="submit">Отправить<
    

        При использовании этих элементов также надо учитывать, что Firefox и IE11, не говоря уже о более старших версиях IE,
        не поддерживают эти элементы.
