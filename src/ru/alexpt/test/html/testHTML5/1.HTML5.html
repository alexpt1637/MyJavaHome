Кнопка
Элемент div имеет атрибут style. После знака равно в кавычках пишется значение атрибута ("color:red;")
Элемент input, состоит из одного тега и имеет два атрибута: type (указывает на тип элемента - кнопка) и value (определяет текст кнопки)
Атрибут disabled указывает, что данный элемент отключен
