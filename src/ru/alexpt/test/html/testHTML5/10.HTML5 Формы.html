Формы в html представляют один из способов для ввода и отправки данных.
        Все поля формы помещаются между тегами 'form' и '/form'. Например, создадим простейшую форму:
    
        Для настройки форм у элемента form определены следующие атрибуты:
        method: устанавливает метод отправки данных на сервер. Допустимы два значения: post и get.

        Значение post позволяет передать данные на веб-сервер через специальные заголовки.
        А значение get позволяет передать данные через строку запроса.

        action: устанавливает адрес, на который передаются данные формы
        enctype: устанавливает тип передаваемых данных.

        Он свою очередь может принимать следующие значения:
        application/x-www-form-urlencoded: кодировка отправляемых данных по умолчанию
        multipart/form-data: эта кодировка применяется при отправке файлов
        text/plain: эта кодировка применяется при отправке простой текстовой информации

        'form method="post" action="https://www.farpost.ru/vladivostok/auto/moto/sale/+/Honda+ST+1300/"'
            'input name="login"/'
            'input type="submit" value="Войти" /'
        '/form'

        В выше использованном примере: у формы установлен метод "post", то есть все значения формы отправляются в теле запроса,
        а адресом служит строка. Как правило, по указанному адресу работает веб-сервер, который, используя одну из технологий
        серверной стороны (PHP, NodeJS, ASP.NET и т.д.), может получать запросы и возвращать ответ.
        В данном же случае мы не будем акцентировать внимание на технологиях серверной стороны, сосредоточимся лишь на тех
        средствах HTML, которые позволяют отправлять данные на сервер.
